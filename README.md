## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio da **DIO (Digital Innovation One)** e tem como objetivo construir um sistema baseado em **Agentes de IA** para auxiliar usuários na descoberta de carreiras em tecnologia.

A solução simula um fluxo real de decisão utilizando dois agentes especializados:

- 🎯 **Agent 1 – Entrevistador de Carreira**
- 🧠 **Agent 2 – Planejador de Roadmap**

---

## 🧠 Como Funciona

O sistema segue um fluxo estruturado:

1. O usuário responde perguntas sobre perfil e interesses  
2. O Agent 1 analisa as respostas e sugere carreiras  
3. O usuário escolhe uma carreira  
4. O Agent 2 gera um roadmap completo de estudos  

---

## 🤖 Arquitetura dos Agentes

### 🔹 Fluxo de Execução

    Usuário
       ↓
    Agent 1 (Entrevista)
       ↓
    Análise de Perfil
       ↓
    Sugestão de Carreiras
       ↓
    Escolha do Usuário
       ↓
    Agent 2 (Planejamento)
       ↓
    Roadmap Personalizado

---

## 🧩 Agent 1 – Entrevistador de Carreira

Responsável por:

- Coletar informações do usuário
- Entender perfil técnico e comportamental
- Identificar interesses e objetivos
- Sugerir carreiras alinhadas

### 📊 Critérios de análise:

- Interesses
- Experiência
- Tempo disponível
- Objetivo profissional
- Preferências (dados, código, pessoas)

---

## 🧠 Agent 2 – Planejador de Carreira

Responsável por:

- Gerar roadmap personalizado
- Definir plano de estudos (90 dias)
- Sugerir projeto de portfólio
- Preparar usuário para entrevistas

### 📦 Entregáveis:

- Visão do dia a dia da carreira
- Mapa de skills
- Roadmap estruturado
- Projeto prático
- Roteiro de entrevistas
